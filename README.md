# AHSS（Animal Health Support System）

動物園向け・日次バイタルチェック支援ツール（Excel VBA × Python 可視化）

------------------------------------------------------------------------

## 概要

AHSSは、動物園の飼育員が日々入力する体温・食欲などのバイタル情報をもとに、\
**入力ミスの即時検出** と **異常傾向の可視化**
を行う業務支援ツールです。

Excel
VBAで入力検証・異常判定・CSV出力を行い、Pythonで分析グラフを生成し、\
結果をExcel上で確認できます。

------------------------------------------------------------------------

## 想定ユーザー

-   飼育員：バイタルデータ入力・チェック
-   班長（管理者）：監査ログ確認・ダブルチェック
-   獣医師：異常傾向の分析・予防的観察

------------------------------------------------------------------------

## 主な機能

### CSV出力

-   入力値の妥当性チェック
-   基準値に基づく異常判定
-   AlertLog記録
-   alert_log.csv生成

### データ分析

PythonでCSVを分析し、以下のグラフを生成：

-   種別別件数
-   異常理由別件数
-   種別 × 異常理由（積み上げ）

### 分析データ表示

生成されたPNGグラフをExcelのGraphシートに表示

### 入力業務支援

Inputシートへの入力を支援

------------------------------------------------------------------------

## 使用技術

-   Excel VBA（業務ロジック）
-   Python（matplotlib）（データ分析・可視化）
-   CSV（システム連携）

------------------------------------------------------------------------

## 実行方法

### 1. Excelを開く

AHSS.xlsmを開き、マクロを有効化

### 2. MAINシートで操作

ボタンを順に押す

1.  CSV出力
2.  データ分析
3.  分析データ表示
